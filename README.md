# 🍑 Booty Deluxe Bot

A Telegram bot that grows a seed into a pixelated booty via ASCII animation.

## Commands
- `/start` — Welcome message
- `/grow` — Plant a seed and watch it bloom

## Deploy to Railway
1. Push this repo to GitHub
2. Connect it to [Railway](https://railway.app)
3. Add environment variable: `BOT_TOKEN` = your Telegram bot token
4. Deploy

## Run Locally
```bash
pip install -r requirements.txt
BOT_TOKEN=your_token_here python booty_deluxe_bot.py
```
